# Dotfiles

Run

```bash
sudo apt update
sudo apt install -y git curl sudo python3 python3-venv tar
curl -fLSso /tmp/yadm https://github.com/TheLocehiliosan/yadm/raw/master/yadm
chmod +x /tmp/yadm
sudo mv /tmp/yadm /usr/local/bin/yadm
yadm clone --bootstrap https://github.com/minaboktor2628/dotfiles.git

```
